nu.css

A classless CSS Framework

Very useful information!

"Nu" is used in dialects from the Dresden region and simply means "yes". Nu is, of course, the answer to every quality aspect of a CSS framework!
  • Is it accessible? Nu.
  • Is it responsive? Nu.
  • Is it simplistic? Nu.
  • Is it compatible? Nu.
  • Is it extensible? Nu.
  • Is it classless? Nu nu.
If you would like to read more about how "nu" is used in these dialects, consider reading the Wikipedia page on "Nu".

So why is it not called "arscec"?

It reminded me of another German word I didn't find suitable: Arschkeks.

Usage tutorial

Simply run the following piece of shell code to download and compile the framework:
git clone https://github.com/nkreer/nu && cd nu && ./build.sh You may also click on to do things!
